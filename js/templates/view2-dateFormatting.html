Date formatting

Default application date format:
  • year: "numeric",
  • month: "long",
  • day: "numeric",
  • hour: "numeric",
  • minute: "numeric",
  • second: "numeric"
scope values in controller:
  • $scope.date = new Date();
Description HTML code Result
Date (default app format) <span ng-bind="date | a9n_dateFormat"></span>
Date filter (default format) <span ng-bind="date | a9n_dateFormat:{}"></span>
Date filter (es locale, default app format) <span ng-bind="date | a9n_dateFormat:{locale:'es'}"></span>
Date filter (custom format) <span ng-bind="date | a9n_dateFormat:{year: 'numeric', month: 'short', day: 'numeric'}"></span>
Date directive <span a9n-date-format="date"></span>
Date directive (default format) <span a9n-date-format="date" format-options="{}"></span>
Date filter (es locale, default app format) <span a9n-date-format="date" format-locale="es"></span>
Date directive (custom format) <span a9n-date-format="date" format-options="{year: 'numeric', month: 'short', day: 'numeric'}"></span>
